# Tapestry Cards - Single-sided

Order of Elements: `A F E W`
so that Fire and Water (and Air/Earth) are not (usually) near to each other
on the cards.

Total number of cards = 66 = 3x4 + 7x6 + 3x4

## 3

```
+---------+  +---------+  +---------+
|    X  X |  | X     X |  | X  X    |
| X       |  |    X    |  |       X |
+---------+  +---------+  +---------+
```
4 copies of each card

3 blank spots: a,b,c:

* a:  A  W  E  F
* b:  F  A  W  E
* c:  E  F  A  W
